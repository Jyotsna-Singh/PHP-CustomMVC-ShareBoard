# ShareBoard

ShareBoard is an Idea Sharing Application built with a Custom MVC PHP Framework.

* Features/Technologies: 
  * PHP Syntax, PHPMyAdmin, MySQL
  * HTML/CSS
  * Bootstrap
  * RDBMS within OOPS framework
